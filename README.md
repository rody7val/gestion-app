Entorno de gestión de aplicaciones de escritorio

### Instalación
```
git clone https://github.com/rody7val/gestion-app
npm install

```

### Desarrollo

```
npm run dev
```

### Producción

```
npm run build
npm run electron
```

### Empaquetado

windows:
```
npm run package-win
```

linux:
```
npm run package-linux
```

mac:
```
npm run package-mac
```
