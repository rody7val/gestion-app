Entorno de gestión de aplicaciones de escritorio

### Instalación
```
git clone https://github.com/rody7val/gestion-app
npm install

```

### Desarrollo

```
npm run dev

```

### Producción

```
npm run build
npm run electron

```

### Empaquetado

windows:

```
package-win

```

linux:

package-linux
